# Challenge 4 — Detect Single-Character XOR

**Link:** https://cryptopals.com/sets/1/challenges/4

## What it does
One of 326 hex-encoded lines was encrypted with single-byte XOR. Find which one.

## Key insight
You don't need to know which line is encrypted beforehand — you run `crack_single_byte_xor` on every line and compare their best scores. The encrypted line will produce a decryption attempt that looks far more like English than any of the other 325 random-looking lines. The score itself is the filter.

This is a common pattern in cryptanalysis: apply an attack to many candidates, let the scoring function do the discrimination.

## Why it matters
No new crypto here — the entire solution reuses ch01 and ch03. This challenge is about applying an existing attack at scale, which is how real-world cryptanalysis works. The detected message is "Now that the party is jumping", key `0x35`.

## Functions
- `detect_single_byte_xor()` — reads a file of hex lines, cracks each, returns the best overall result
